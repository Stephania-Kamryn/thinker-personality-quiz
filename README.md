# Thinker Personality Quiz

An interactive web-based personality quiz that determines what type of thinker a user is based on their responses to a set of questions.

## Overview

This project presents a series of multiple-choice questions designed to evaluate how a user approaches problems and decision making. Based on the selected responses, the application calculates a score and classifies the user into one of four thinker types.

The result is displayed in a modal window after submission.

## Features

- 10-question personality quiz
- Score calculation using JavaScript
- Modal popup displaying the result
- Form validation to ensure all questions are answered
- Responsive card-style layout using Tailwind CSS
- Option to retry the quiz

## Technologies Used

- HTML
- Tailwind CSS
- JavaScript

## Thinker Types

The quiz categorizes users into the following types:

- Analyst
- Innovator
- Connector
- Adventurer
